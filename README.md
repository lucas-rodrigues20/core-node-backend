# NodeJs Back-End Core

* Run `npm install` to get the dependencies;
* Create and fill a `.env` file in the root directory with this structure:
  ```
    DB_USER=""
    DB_PASSWORD=""
    DB_NAME=""
    DB_TEST_NAME=""
    LOG_LEVEL=""
    MAIL_SERVICE=""
    MAIL_USER=""
    MAIL_PASSWORD=""
  ```
* Run `npm start` to start the server on port 3000.
